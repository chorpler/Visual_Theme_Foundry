Feedback & State

Alerts, toast notifications, spinners, progress bars, tooltips, and notification badges — semantic color as status signal throughout the UI lifecycle.

Info — primary pair Informational messages use primaryContainer. High visibility, neutral emotional valence.
Success — tertiary pair Tertiary carries success meaning in M3. Harmonically separate from the primary action color.
Warning — secondary pair Secondary as caution signal — softer than error, more urgent than info. Context-dependent.
Danger — error pair Error has a first-class semantic role in M3. Use errorContainer for non-destructive warning states.
